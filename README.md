Dashboard de Análisis de Lanzamientos de SpaceX
Proyecto desarrollado como parte de mi especialización en Data Science y Machine Learning.

Objetivo: Analizar la relación entre la carga útil (Payload) y el éxito de los lanzamientos de SpaceX mediante visualizaciones interactivas.

Tecnologías utilizadas:

Python para el procesamiento de datos.

Dash & Plotly para la interfaz interactiva y gráficos dinámicos.

Pandas para el manejo del dataset.
